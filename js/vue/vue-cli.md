# vue-cli

* [文档](https://cli.vuejs.org/zh/guide/)

* vue-cli npm中有两个版本
  * `npm install -g vue-cli` 2.X
  * `npm install -g @vue/cli` 4.X

## 新版

* `vue ui` 启动可视化界面管理项目
  * 创建管理项目
  * vue-cli插件管理，搜索、新增
  * npm命令运行和配置
* @vue/cli-service 安装了一个名为 `vue-cli-service` 的命令
  * vue-cli插件也会注入额外的命令
  * `npx vue-cli-service help [command]` 查看所有注入命令或单个命令
